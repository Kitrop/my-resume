База данных для сервисного центра

Курсовая работа в группе из 5 человек. Организовавал работу в команде, создал логическую и физическую модель базы данных. Проект основан на документах официального дилерского центра. Было создано более 30 таблиц и более 20 запросов.

Сайт для веб-студии

Создал логику сайта для открывающейся веб-студии. Суть заключалась в создании простого, но визуально красивого сайта, с поддержкой чат-бота для менеджеров для отслеживания заявок. Принимал макеты, руководил верстальщиком, в свою очередь занимался логикой сайта и чат-ботом.

Мастер-класс по созданию Web-app telegram

Участвовал на мастер-классе на мероприятии "я ❤️ frontend", где показывали, что представляет из себя telegram web apps. После чего вдохновился и создал цветочный магазин на telegram web app.

Проекты

  1. Video Bot

    Telegram bot разработнный для просмотра и загрузки видео прямо в телеграм. К телеграм боту подключенны онлайн кассы: Юкасса и Робокасса. Также имеется рекуретный платеж.

    Nest.js • Telegraf • TypeScript • PrismaORM • PostgreSQL

  2. Electronic-shop API

    Api для магазина электронники с возможностью добавления пользователей/товаров/отзывов, а также их изменения
    GitHub: https://github.com/Kitrop/electronic-shop

    Nest.js • TypeScript • Bcrypt • PrismaORM • PostgreSQL • JWT

  3. Webchat-NestJS

    Чат с возможностью указать свое имя и текст сообщения, сообщения приходят моментально и синхронизируется со всеми пользователямию. Чат написан с использованием socket.io основного на WebSocket. Отрисовка интерфейса была осуществеленна с помощью ejs
    Github: https://github.com/Kitrop/NesJS-webChat

    Socket.io • Express • Nest

  4. Space-adventure

    Текстовая игра для телеграм бота, о путешествии в космосе, где игрок должен собирать ресурсы, бороться с врагами и захватывать планеты. Игра находится в разрботке
    GitHub: https://github.com/Kitrop/spaceAdventure

    React • TypeScript • Redux

  5. Social-React

    Веб-приложение в котором имеются профили пользователей, можно писать сообщения в общий чат, добавлять посты, а также добавлять людей в друзья
    Github: https://github.com/Kitrop/ReactSocialNetwor

    React • Redux • TypeScript • Axios

  6. CartNext

    Интернет-магазин основанный на Next.js с возможностью менять тему, корзина реализована на модальном окне
    Github: https://github.com/Kitrop/cartNext

    Next • Typescript • Styled-components

  7. FlowerStore

    Telegram web app, приложение в телеграм для выбора цветов, есть возможность добавления цветов в корзину с последующим заказом. Под функционал написаны тесты на jest
    Github: https://github.com/Kitrop/flowerStore

    JavaScript • Telegram API • Jest

  8. RecipeFood React

    Веб-приложения для поиска рецептов, есть возможность указывать требования, такие как калории, время приготовления
    GitHub: https://github.com/Kitrop/recipeFoodReact

    React • TypeScript • Redux
